# Data Science & Machine Learning – YouTube Lesson Code

This repository contains Python code used in my YouTube tutorials covering practical data science and machine learning concepts.

The goal of these lessons is to demonstrate how common analytical and machine learning techniques are implemented in Python using real-world style workflows.

The repository includes examples focused on data exploration, regression modeling, and classification methods such as decision trees.

---

## Topics Covered

The tutorials in this repository cover:

- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Feature engineering
- Regression modeling
- Decision Trees for classification
- Model evaluation
- Visualization of model results

These examples are designed to be simple, practical demonstrations that help learners understand how these techniques are applied in real data science projects.

---

## Technologies Used

- Python
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
